MemTrim Lite now checks for updates in the background and can hand off the latest installer directly from inside the app. The installer upgrades the existing installation in place instead of requiring a manual uninstall first.

For public Windows releases, the cleanest installer experience still requires a trusted code-signing certificate. Without code signing and reputation, Windows SmartScreen may still show a warning even when the build is legitimate.
