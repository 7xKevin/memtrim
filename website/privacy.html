Privacy summary

MemTrim Lite is a local Windows utility. It does not require an account and does not upload memory statistics or GPU statistics to a cloud service.

The app now performs a lightweight background update check by requesting a small version manifest. No personal account data is required for that check.

The website is static. If you deploy it on Render or another host, that hosting provider may collect ordinary web logs such as IP address, browser, and request metadata.
