Simple, fast memory cleanup for Windows.

MemTrim Lite is a lightweight native desktop application built with Win32 and C++. It shows live RAM and GPU statistics, stays available in the tray, and performs safe memory cleanup without unnecessary overhead.

Main features

  • Physical memory, pagefile, and working set statistics
  • GPU adapter name and memory usage view
  • Live updates every second
  • Tray icon with background operation
  • Optional administrator mode for stronger cleanup

Why use it

  • Native Win32 application with no web runtime
  • Compact fixed-size interface
  • Fast startup and low memory overhead
  • Installer and portable builds available

Installation

  1. Download MemTrimLite-Setup.exe.
  2. Run the installer and complete the setup wizard.
  3. Launch the application from Start Menu or desktop shortcut.
  4. Use the tray icon to reopen it when it is running in the background.
